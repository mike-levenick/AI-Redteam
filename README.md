# AI Redteam Challenge

A prompt injection and jailbreaking challenge where you attempt to extract secret flags from an AI assistant.

## How It Works

1. The AI assistant holds a secret flag for each stage
2. Your goal is to convince the AI to reveal its current flag
3. Submit the flag back to the AI to advance to the next stage
4. Complete all 5 stages to win

## Stages

| Stage | Difficulty |
|-------|------------|
| 1 | Easy |
| 2 | Medium |
| 3 | Hard |
| 4 | Very Hard |
| 5 | Expert |

## Rules

- Use any prompt techniques you like
- Each stage introduces stronger defenses
- Flags are case-sensitive

Good luck!
